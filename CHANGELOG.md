# Changelog

## v1.0.0 — Initial Release

### Wireless baseband

- QPSK, 16-QAM, and 64-QAM modulation
- OFDM transmitter and receiver
- Cyclic-prefix processing
- AWGN and flat Rayleigh fading
- Narrowband interference
- Pilot-based channel estimation
- One-tap equalization
- BER, EVM, and pilot-error measurements

### Machine learning

- Adaptive modulation selection
- Receiver fault classification
- Cached model inference
- Generated training datasets and evaluation reports

### Verification

- Unit tests
- Integration tests
- Fault-injection tests
- GitHub Actions continuous integration

### Deployment

- Raspberry Pi 4 deployment
- Streamlit dashboard
- Automatic startup through systemd
- Raspberry Pi performance benchmarks
- One-command end-to-end demo
